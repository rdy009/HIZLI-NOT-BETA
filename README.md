# HIZLI-NOT-BETA📝 Hızlı Not - Açık Kaynak Not Alma Uygulaması

Pardus ve Debian tabanlı Linux dağıtımları için Python ve PyQt5 kullanılarak geliştirilmiş, hafif ve kullanıcı dostu bir not alma uygulamasıdır.
✨ Özellikler

    Hızlı Not Alma: Sade arayüzü ile anında notlarınızı yazın.

    TXT Desteği: Notlarınızı standart metin formatında kaydedin.

    PDF Desteği: Notlarınızı paylaşılabilir PDF formatında dışa aktarın.

    Açık Kaynak: Tamamen şeffaf ve geliştirilebilir kod yapısı.

🚀 Kurulum

Uygulamayı sisteminize kurmak için .deb paketini indirmeniz yeterlidir. Bağımlılıkların otomatik olarak çözülmesi için terminal üzerinden apt ile kurmanız önerilir.

    Terminali açın.

    .deb paketinin bulunduğu klasöre gidin.

    Aşağıdaki komutu çalıştırın:

Bash

sudo apt install ./hizli-not.deb

🛠 Çalıştırma

Kurulum tamamlandıktan sonra uygulamayı iki şekilde başlatabilirsiniz:

    Uygulama Menüsü: Uygulama listenizde "Hızlı Not" şeklinde aratın ve tıklayın.

    Terminal: Terminale hizli-not yazarak uygulamayı başlatın.

📦 Bağımlılıklar

Uygulamanın çalışması için sisteminizde aşağıdaki paketlerin bulunması gerekir (Kurulum sırasında otomatik yüklenir):

    python3

    python3-pyqt5

    python3-reportlab (PDF desteği için)
